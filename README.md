## ✨ Overview

**SkyCode Tools** is a modern full-stack web application that provides simple, fast, and free online utilities for students, developers, and professionals.

It helps users perform daily file and image tasks directly in the browser without installing any software.

### 🎯 Main Goals
- Fast file processing
- Clean and modern UI
- Easy-to-use online tools
- Secure and responsive experience
- Free accessibility for everyone

---

## 🛠️ Features

### 📄 File Tools
- 🖼️ Image to PDF
- 📚 Merge PDF
- ✂️ Split PDF

### 🖼️ Image Tools
- 🗜️ Image Compressor
- 🎨 Background Remover

### 💡 Extra Features
- Drag & Drop Upload UI
- Animated Progress Bar
- Download Processed Files
- Smooth Hover Animations
- Responsive Design
- Modern Dark + Blue Gradient UI
